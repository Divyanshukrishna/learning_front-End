Selectors in HTML

Element Selector

the element selector selects HTML elements based on the element name.

Id selector

The id selector used the id attribut of an HTML element to select a specific element. The id of an element is unqiue within a page, so the id selector us used to to select one unique element!

Class Selector

The class selector selects HTML elements with a sepecific class attribute. To select elements with a specific class, write a period() character, followed by the class name.
